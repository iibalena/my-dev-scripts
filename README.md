```md
# my-dev-scripts

Coleção de scripts pessoais e utilitários para automação de tarefas em projetos pessoais e corporativos.

## Estrutura

### 📁 migration/
Scripts para reorganização de projetos e repositórios Git.

- `extract-subfolder-to-repo.sh`: extrai uma pasta de um monorepo para um novo repositório mantendo histórico
- `migrate-repo-to-another-account.sh`: clona e reenvia um repositório da conta Ultra para a conta pessoal

### 📁 deploy/
Scripts usados para deploy manual/automatizado de projetos.

- `deploy-personal-app.sh`: deploy de projeto pessoal (ajuste conforme necessidade)

## Requisitos

- Git
- Bash
- (opcional) `git-filter-repo` para migração de histórico
```
